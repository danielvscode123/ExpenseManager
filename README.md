
# Finance Palace App

Este projeto é um painel financeiro moderno feito com React, Vite, TypeScript, shadcn-ui e Tailwind CSS.

## Como rodar localmente

1. Clone o repositório:
	```sh
	git clone <YOUR_GIT_URL>
	cd <YOUR_PROJECT_NAME>
	```
2. Instale as dependências:
	```sh
	npm install
	```
3. Inicie o servidor de desenvolvimento:
	```sh
	npm run dev
	```

## Tecnologias utilizadas
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Para publicar, utilize sua plataforma de preferência (Vercel, Netlify, Firebase Hosting, etc). Basta buildar com:
```sh
npm run build
```
E subir o conteúdo da pasta `dist/`.

## Domínio personalizado

Para conectar um domínio, siga as instruções da plataforma de deploy escolhida.
